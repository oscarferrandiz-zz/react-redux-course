# Create-react-app

## 1. Initializing app

* `npm install -g create-react-app`
* `create-react-app my-app`
* `cd my-app/`
* Create a ‘.env’ file at the root level (same level as package.json)
* Set an environment variable, `NODE_PATH` to `src/`
* `npm start` or `yarn start`
